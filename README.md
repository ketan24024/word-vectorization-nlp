# Word Vectorization NLP Project

## 📌 Objective
To compare Bag of Words and TF-IDF vectorization techniques for sentiment analysis.

## 📊 Dataset
Sentiment140 Twitter dataset

## ⚙️ Methods Used
- Bag of Words (BoW)
- TF-IDF
- Logistic Regression
- Support Vector Machine (SVM)

## 📈 Results
- BoW + LR Accuracy: 0.7075
- TF-IDF + LR Accuracy: 0.7115
- TF-IDF + SVM Accuracy: 0.7000

## 🔍 Conclusion
TF-IDF with Logistic Regression performed best by providing better feature weighting and improved accuracy.
